# ftpclient module #

 * Installs package hj-ftptool if not installed.
 * Installs package ftp if not installed.
 * Places /etc/profile.d/local.FTP_PASSIVE.sh, which sets FTP_PASSIVE=1 for ftp clients that honor it and use /etc/profile.d/.

# Compatibility #

Note that this module requires packages specific to Herff Jones and is not suitable for other parties.

 * EL 6

# Parameters #

Not applicable
